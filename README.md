## jumps, marks (from usr_03)
`:ju[mps]` <br />
`:cle[arjumps]` <br />
`m'` <br />

另外跳行操作`:n`和`nG`(这个n代表你要跳到哪一行) <br />
只有`nG`会把将原来的位置加入jump list <br /> 

`'"`			跳到上次编辑退出时候的光标位置 <br />
`'[`			最后一次进入编辑模式的地方 <br />
`']`			最后一次退出编辑模式的地方 <br />
